# 网络调研技能

当用户需要搜索互联网获取最新信息时，按以下流程执行：

## 步骤

1. **分解问题**：将用户的调研需求分解为 2-4 个具体搜索关键词
2. **执行搜索**：调用 `web_search` 工具，使用不同的关键词组合
3. **筛选结果**：
   - 优先选择权威来源（官方网站、学术论文、知名媒体）
   - 过滤广告和低质量内容
   - 选取最相关的 3-5 条结果
4. **深入阅读**（可选）：
   - 对重要来源调用 `fetch_url` 获取详细内容
   - 提取关键数据和观点
5. **整理报告**：
   - 按主题组织信息
   - 标注每条信息的来源 URL
   - 总结关键发现和趋势
   - 指出信息的时效性和局限性

## 注意事项

- 明确区分事实和观点
- 标注信息来源，方便用户验证
- 如果搜索结果矛盾，都要呈现并说明
- 注意信息的时效性，标注发布日期
