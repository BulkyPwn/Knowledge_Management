# 搜索导入技能

当用户希望搜索互联网并将结果导入知识库时，按以下流程执行：

## 步骤

1. **搜索内容**：调用 `web_search` 工具搜索用户指定的关键词
2. **展示结果**：将搜索结果以列表形式展示给用户，包含标题、URL 和摘要
3. **等待选择**：询问用户要导入哪些结果（可以建议全部导入或选择性导入）
4. **抓取内容**：对用户选中的每条结果，调用 `fetch_url` 获取完整内容
5. **内容处理**：
   - 提取核心文本内容，去除导航、广告等无关部分
   - 检查内容质量（是否被付费墙拦截、是否为空等）
6. **报告结果**：
   - 告知用户成功抓取了多少条内容
   - 展示每条内容的标题和字数摘要
   - 建议用户使用知识库管理界面完成最终导入

## 注意事项

- 抓取失败时提供具体的错误原因（网络超时、403禁止访问等）
- 对于 PDF 链接，提醒用户可能需要单独处理
- 尊重 robots.txt 和网站的使用条款
