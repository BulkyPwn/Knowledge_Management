# 知识库问答技能

当用户提问涉及知识库内容时，按以下流程执行：

## 步骤

1. **理解问题**：分析用户问题的核心意图，提取关键词
2. **检索知识库**：调用 `knowledge_query` 工具搜索相关内容
3. **评估结果**：
   - 如果结果充分 → 直接回答问题，引用来源
   - 如果结果不足 → 尝试不同关键词重新搜索
   - 如果完全无结果 → 告知用户并建议尝试其他问题
4. **组织回答**：
   - 使用清晰的结构化格式
   - 引用来源文件路径，格式为 `[来源名]`
   - 如果多个来源有不同观点，都要呈现
   - 明确区分"知识库中的内容"和"你的推理/补充"

## 注意事项

- 优先使用知识库中的信息，不要编造不存在的内容
- 如果知识库内容有限，明确告知用户这是基于现有资料的回答
- 对于技术问题，尽量引用具体的文件路径
