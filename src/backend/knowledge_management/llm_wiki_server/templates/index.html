LLM 模型配置

此配置将同步到 KMA 后端，用于 AI 摄入、语义分析、Embedding 向量化等功能。

配置说明

配置保存后会自动同步到 KMA 后端的 app-state.json，使其对所有 AI 功能生效。

API 端点 URL：支持所有兼容 OpenAI Chat Completions 格式的 API，例如：

  • https://api.openai.com/v1 — OpenAI 官方
  • https://api.deepseek.com/v1 — DeepSeek
  • https://dashscope.aliyuncs.com/compatible-mode/v1 — 阿里百炼

API Key：在相应平台获取的密钥。

Embedding 模型：用于 Rescan 时将文本转为向量，留空则使用 Chat 端点自动处理（部分 API 不支持，建议填写专用 embedding 模型）。

⚠ 配置将保存到 config/llm_config.json 文件，不会上传到任何远程服务。
